botの作成方法

1. このサーバーに参加してください　Storm

2. #lobby-bot のチャンネルの所で「+startbot」と入力してください

3. Storm Lobby Bots からDMが届くので「• Go to: https://...」と書いてる所のリンクをクリックしてください

4. 使わないepicアカウントか新規アカウントでログインして「確認」をクリックしてください

5. 「Lobby Bot Ready as: [アカウント名]」と書かれていれば作成完了です

6. botのアカウントにフレンドを送信してください

7. botのパーティーに招待するか参加してください
※パーティーに招待する場合は招待を送るとDiscordで「Party Invite From ...」と送られてくるので✅を押すと参加できます。

Botは60分でオフラインになります。
再度使用する場合は上記の botの作成方法 の2から行ってください。

コマンド一覧

コマンド 説明 使用例
+skin name スキンの変更 +skin cuddle team leader
+emote name エモートの変更 +emote hey now!
+backpack name バックアクセサリーの変更 +backpack fabled cape
+pickaxe name ツルハシの変更 +pickaxe balloon axe
+level number レベルの変更 +level 200
+styles cosmetic name スタイルの変更 +style tomatohead / 2
+hide プレイヤーを非表示 (botがリーダー時のみ有効) +hide
+unhide プレイヤー非表示を解除 (botがリーダー時のみ有効) +unhide
+ready 準備ok に変更 (試合は無効) +ready
+unready 準備ok 状態を解除 +unready
+send message パーティーにメッセージを送信 +unready
+privacy public パーティーを パブリック に変更 (botがリーダー時のみ有効) +privacy public
+privacy private パーティーを プライベート に変更 (botがリーダー時のみ有効) +privacy private
+sitin 準備中 に変更 +sitin
+sitout 欠場中 に変更　--現在は無効？-- +sitout
+match number botをマッチ状態に変更　(1～255の範囲で可能) +match 120
+unmatch botをマッチから帰還させる　--現在は無効？-- +unmatch
+help Discordにコマンドが送られてくる +help
+stop botを停止させる +stop
